# Growell - Platform Parenting Digital Indonesia

Landing page untuk platform parenting digital Growell yang membantu orang tua memantau tumbuh kembang anak.

## Teknologi

- React 18
- Vite
- Tailwind CSS (via CDN)
- Lucide React (Icons)

## Instalasi

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Build untuk production:
```bash
npm run build
```

4. Preview build production:
```bash
npm run preview
```

## Fitur

- Landing page responsif dengan desain modern
- Navigation dengan mobile menu
- Hero section dengan CTA
- Features section
- How it works section
- Testimonials
- FAQ section
- Footer dengan links
